
# 项目本地访问地址

目前是本机ip地址加上线上路径并且需要拼接memberSid， Sid可以在store文件夹下获得
示例：
https://10.50.10.11/h5-static/member-compliance/#/?memberSid=42c4ba8544ac40b18fc364cb613573bb
